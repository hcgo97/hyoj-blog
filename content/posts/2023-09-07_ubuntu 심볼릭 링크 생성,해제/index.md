### 심볼릭 링크란?
→ 윈도우의 바로가기와 동일

### 심볼릭 링크 생성
```
ln -s [원본 경로] [링크 경로]
```

### 심볼릭 링크 삭제
- rm 명령어 사용해도 되지만, 만약 원본 경로가 디렉토리라면 원본 내부의 파일을 삭제해버릴수도 있기 때문에 unlink 사용 권장

```
unlink [링크 경로 or 심볼릭 링크 이름]
```
