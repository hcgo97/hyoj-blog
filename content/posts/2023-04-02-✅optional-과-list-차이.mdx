# Optional List

### Optional\<T\>

- ***
- , null
- **ifPresent()** : x
  Optional (null x)
- **isPresent()** : x
  Optional null
  if-else null

### List\<\>

- ***
- null Empty List

수형 인자

- **isPresent()** : 특정결과 반환x

  if-else문 안쓰고 null값 검사 가능
