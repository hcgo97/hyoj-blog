# Postman Test

### responseBody equals test

```jsx
// responseBody json
const jsonData = pm.response.json()

// pm.test(, )
pm.test("responseZoneId is 'Z009'", () =\> \{

		// pm.expect().to.eql()
		pm.expect(jsonData.data.responseZoneId).to.eql("Z009");
\}
```

### status

const status = pm.response.status // "OK"

*/
console.log(trueData[0]);
