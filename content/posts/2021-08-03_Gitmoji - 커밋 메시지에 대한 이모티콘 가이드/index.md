# Gitmoji 란?

- github 커밋 메시지에 **이모지 사용을 표준화하고 설명**하기 위한 구상입니다.
- 커밋 메시지에 이모티콘을 사용하면 사용된 이모티콘만 보고 **커밋의 목적이나 의도를 쉽게 식별**할 수 있습니다.
- 링크: [https://gitmoji.dev](https://gitmoji.dev/)
