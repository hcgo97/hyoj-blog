### 3. const (block scope)

- 상수 - 재할당 불가, 재선언 불가
- 선언과 동시에 반드시 초기화 필요
- 범위는 let과 같음 (블록 유효범위)
- 함수 선언만 hoisting 됨, 함수 표현식은 x

```jsx
/* 함수 선언 */

foo(); // "bar"

function foo() \{
  console.log("bar");
\}

/* 함수 표현식 */

foo(); // TypeError: foo is not a function

var foo = function () \{
  console.log("bar");
\};
```
